# [Bitly Bot](https://telegram.dog/SL_BitlyBot)

Bitly URL shortener bot to shorten Unlimited URLs.

---

## Setup

Collect these env variables to deploy this bot:-

* `API_ID`
* `API_HASH`
* `BOT_TOKEN`
